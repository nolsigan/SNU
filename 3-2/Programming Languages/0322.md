#### natlist induction proof

* for all l : natlist, P(l) (l에 대한 가설의 증명이 존재한다)
* P(nil) 증명
* for all h, t :  natlist, P(t)가 true일 때 P(h::t)가 true임을 증명


```
Fixpoint nlind
    (P: natlist -> Prop)
    (PB: P nil)
    (PS: forall h t (pf: P t), P (h::t))
    (l : natlist)
    :
    P l
:=
    match l with
    | nil => PB
    | h :: t =>
        PS h t (nlind P PB PS t : P t)
    end.
```
