## Continue Graph Theory

#### Directed Graphs Properties

* In(A), Out(A)
    * 여러개의 edge로 연결 되어 있어도 카운트

#### Clique

* Complete Subset
    * subset이 complete인 애
    * node 하나나 연결된 두개의 node도 clique
* maximal clique
    * 이 clique를 포함하는 더 큰 clique은 없을 때
* maximum clique
    * maximal clique 중 가장 node 수가 많은 clique
